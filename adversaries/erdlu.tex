\subsection{Erdlu}

\Creature[name=Erdlu, type=Rival, brawn=3, agility=3, intellect=1, willpower=1, presence=1, soak=3, wounds=12, meleedef=0, rangeddef=0, magicdef=0, psionicdef=0]{
The erdlu is a smaller version of the crodlu. Its body
is covered in tough scales, and its folded forelimbs
sprout useless wings. Hardy and fast, this drakelike
creature is a fine riding beast for a Silhoutte 0 rider. It is
too skittish to be trained for war, however.
} {
    % skill
    Athletics 2, Brawl 1
}{
    % talents
    None
}{
    \\ % ability
    \textbf{- Trained Mount:} Add \boost to a rider's Riding check while mounted\\
    \textbf{- Beast of Burden 5:} Add 5 to encumbrance threshold\\
    \textbf{- Sure-Footed:} Erdlu suffer no penalties while moving through difficult terrain\\
    \textbf{- Lightning Charge:} Erdlu can sprint extremely fast.\\
}{
    \\ % equipment
    \textbf{- Beak:} Brawl; Damage: 5; Critical: 5; Range [Engaged]\\
}
