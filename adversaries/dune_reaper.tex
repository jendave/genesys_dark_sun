\subsection{Dune Reaper}

\Creature[name=Dune Reaper Drone, type=Rival, brawn=4, agility=3, willpower=3, presence=1, soak=5, wounds=14, strain=14, meleedef=2, rangeddef=2, magicdef=0, psionicdef=1]
{
    % description
    A chorus of wailing howls and shrieks and a sickly
    sweet odor on the night breeze warn a traveler that a
    dune reaper pack is on the hunt nearby. By the time
    luminescent eyes gleam like floating lanterns against
    the darkened dunes, escape is no longer possible.
}{
    % skill
    Brawl 2, Cool 2, Perception 3, Stealth 4, 
}{
    % talents
    None
}{
    \\ % ability
    \textbf{- Dark Vision: } A Dune Reaper can ignore \setback\setback for darkness from preception check related to vision.\\
    \textbf{- Shrieking Leap: } Folding its legs beneath its body, a Dune Reaper has an impressive leap. Each round, as a
        manouver, a Dune Reaper can move anywhere up to medium range vertically or up short range vertically. If it attack
        in the same round, it gains a \boost as if it also took the aim manouvre.\\
}{
    \\ % equipment
    \textbf{- Arm Blades: } Brawl; Damage: 6; Critical: 2; Range [Engaged], \iqtyref{accurate} 1\\
}
